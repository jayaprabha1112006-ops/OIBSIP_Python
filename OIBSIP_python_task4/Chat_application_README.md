## Objective
Implement a real-time chat system where multiple clients can exchange messages via a server.

## Steps Performed
1. Built a command line client and server using Python socket programming.
2. Used multithreading to handle simultaneous client connections.
3. Enabled sending and receiving messages in real time.
4. Added basic input validation and error handling.

## Instructions for Use
1. Open a terminal and navigate to the project directory containing `server.py` and `client.py`.  
2. Start the server : python server.py
3. Open separate terminals for each client and run : python client.py
4. Enter a username when prompted, then start sending messages.
5. Messages sent from one client will appear for all other connected clients,The server must be running before clients connect

## Tools Used
- Python
- Socket programming
- Multithreading

## Outcome
A functional command line chat application where multiple clients can connect to a server and communicate in real time.
