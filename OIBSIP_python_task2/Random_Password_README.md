# Task 2: Password Generator

## Objective
Generate secure passwords based on user-defined preferences such as length, letters, numbers, and symbols.

## Steps Performed
1. Created a GUI using Tkinter to allow user input for password criteria.
2. Implemented randomization logic for generating strong passwords.
3. Added clipboard functionality to copy passwords easily.
4. Included input validation for password length and character selection.

## Tools Used
- Python
- Tkinter
- Random library

## Outcome
The application generates secure passwords with user-selected criteria and allows easy copying to the clipboard.
