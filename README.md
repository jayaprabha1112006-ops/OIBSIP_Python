# OIBSIP Python Internship Submission

## Overview
This repository contains all tasks completed for the Oasis Infobyte Python Internship (OIBSIP). The tasks demonstrate practical implementations in Python, including GUI development, API integration, and networking. Each task includes a dedicated folder with source code and a detailed README outlining objectives, methodology, tools used, and outcomes.

---

## Tasks Completed

### 1. BMI Calculator
- **Objective:** Develop a GUI-based application to calculate Body Mass Index (BMI) and classify health status.  
- **Outcome:** Users can input weight and height to receive their BMI value and health category in a user-friendly interface.

### 2. Password Generator
- **Objective:** Create a GUI application to generate secure, random passwords based on user-defined criteria such as length and character types.   
- **Outcome:** Users can generate strong passwords and copy them to the clipboard easily.

### 3. Weather Application
- **Objective:** Develop a GUI-based application to fetch and display real-time weather data for a specified location using OpenWeatherMap API.  
- **Outcome:** Users can view temperature, humidity, wind speed, weather conditions, and icons for any city.
