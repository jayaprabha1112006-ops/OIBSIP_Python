# Task 3: Weather Application

## Objective
Display real-time weather information for any city using the OpenWeatherMap API.

## Steps Performed
1. Built a GUI with Tkinter for city input and unit selection (Celsius/Fahrenheit).
2. Integrated OpenWeatherMap API to fetch live weather data.
3. Displayed temperature, humidity, wind speed, and weather condition.
4. Loaded weather icons dynamically using the Pillow library.
5. Added error handling for invalid city names or API failures.

## Tools Used
- Python
- Tkinter
- Requests library
- Pillow library
- OpenWeatherMap API

## Outcome
The application provides a user-friendly GUI displaying current weather details for any city.
