# Task 1: BMI Calculator

## Objective
Calculate Body Mass Index (BMI) based on user input for weight and height, and display the health category.

## Steps Performed
1. Created a GUI using Tkinter for user input.
2. Implemented the BMI calculation formula: BMI = weight / (height^2).
3. Classified BMI into categories: Underweight, Normal, Overweight, Obese.
4. Added input validation to ensure realistic values.

## Tools Used
- Python
- Tkinter

## Outcome
The application calculates BMI and displays the corresponding health category in a simple and user-friendly GUI.
